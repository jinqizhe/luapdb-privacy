Privacy Policy

Extension Name: LuaPDB Decrypt Extension
Last Updated: April 20, 2026

Summary: This extension does not collect, store, or share any personal data. Stack trace data is only sent to an internal company server upon explicit user action, and is never transmitted to any third party.

1. Introduction

LuaPDB Decrypt Extension is an internal developer tool built exclusively for authorized employees of our organization. It assists engineers in decrypting obfuscated Lua stack traces displayed on internal crash analysis pages.

2. Data We Access

The extension accesses the following data only when the user explicitly clicks the decrypt button:

  • Encrypted Lua stack trace text — the obfuscated stack trace content already visible to the user on the current page.

The extension does not access, read, or process:

  • Personally identifiable information (PII)
  • Browsing history
  • Cookies or authentication tokens
  • Any content outside of the designated internal AppDump pages

3. How Data Is Used

When the user clicks the decrypt button, the selected stack trace text is transmitted to an internal company API server (l36.leihuo.netease.com:12680) for decryption processing. The decrypted result is returned and displayed directly in the browser — replacing the encrypted text on the page.

This data transmission:

  • Occurs only upon explicit user action — no passive or background data collection
  • Is sent exclusively to an internal company server within the organization's private network
  • Is not stored permanently on any server
  • Is not used for analytics, advertising, or any purpose other than decryption

4. Data Sharing

We do not share any data with third parties. The extension does not integrate with any external analytics platforms, advertising networks, or data brokers.

5. User Consent

No data is collected or transmitted without the user's explicit action. The extension only activates when the user clicks the "🔓 LuaPDB Decrypt" button. Users may choose not to use the decrypt feature at any time.

6. Data Retention

The extension itself retains no data. Stack trace content processed by the internal API server is used solely to produce the decrypted output and is not retained after the response is returned.

7. Security

All data is transmitted within the company's internal network. The extension operates only on designated internal pages (unisdk.nie.netease.com/appdump/l36*) and only communicates with the designated internal API endpoint.

8. Audience

This extension is intended solely for internal use by authorized company employees. It is not designed for or directed at the general public or minors.

9. Changes to This Policy

Any updates to this privacy policy will be reflected on this page with an updated "Last Updated" date.

10. Contact

For questions regarding this privacy policy, please contact your internal development or IT team.
